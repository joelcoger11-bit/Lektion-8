CSS

CSS står för Cascading Style Sheet och är ett sätt att ändra utseendet på en hemsida.

CSS kan antingen användas genom:

  • Inline
  • Internal
  • External

Vi kommer att använda inline den här lektionen. Det betyder att vi kommer lägga CSS som ett attribut direkt i html taggar

Se koden i den här html-filen och https://www.w3schools.com/html/html_css.asp för mer information


Uppgifter

1. Gå till katalogen för förra lektionens uppgift i utforskaren och sedan i terminalen

2. Öppna index.html i en texteditor

3. Ändra bakgrundsfärgen genom att lägga in ett attribut på body

4. Kolla så att det fungerade

5. Lägg upp förändringen på github genom att använda terminalen

  1. git add .
  3. git commit -m "Added background color"
  4. git push

6. Ändra stil på h1

7. Ändra stil på en p-tagg

8. Lek runt med olika CSS-inställningar på din hemsida. Googla gärna runt för att hitta roligt att göra.
